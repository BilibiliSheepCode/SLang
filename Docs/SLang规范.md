# SLang语言规范

## 1 变量

#### 1.1 定义：在SLang中，可以更改与读取的内存区域称为变量

#### 1.2 声明：在SLang中，用以下代码可声明一个变量

```
<var_type> <var_name>
```

#### 1.3 操作：在SLang中，变量的操作有更改，读取两种

##### 1.3.1 更改包括：赋值，自增，自减

###### 1.3.1.1 赋值

```
<var_name> = <value>;
```

###### 1.3.1.2 自增自减

```
<var_name> = <var_name> ++ ;// 或--
```



##### 读取包括：取值，取地址
